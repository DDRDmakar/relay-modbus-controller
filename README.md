# eletechsup R4D3B16 board controller
## Introduction
A simple lightweight tool to control relay board via Modbus (RS-485).  
Works on Windows and Linux  
Developed with:  
Rust + FLTK + tokio-modbus  
## How to build program
```
cargo build --release
```
